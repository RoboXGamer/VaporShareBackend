# VaporShare Backend

This is the backend for VaporShare.

VaporShare is a file sharing platform that allows users to share files with each other.

## Tech Stack

- Node.js
- Express
- MongoDB
- TypeScript

## Installation

1. Clone the repository
2. Install dependencies using `pnpm install`
3. Create a `.env` file in the root directory and add all the env variables as shown in the `.env.example` file
4. Run the development server using `pnpm dev`
